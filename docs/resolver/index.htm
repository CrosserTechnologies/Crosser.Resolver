Crosser Technologies - Dependency Resolver

The Dependency Resolver in Crosser is a tiny (~13 kb) and very fast DI/IoC container. Even though it was written exclusively for Crosser the Resolver can be used without the communication framework. The Crosser Dependency Resolver is what makes it possible override the default modules (or create new modules) in the Crosser Server.

Benchmarks

We used the IoCPerformance Benchmarks from Daniel Palme

The Crosser Dependency Resolver was (as already mentioned) written for the needs in the Crosser Framework. The Resolver does not have all features and functions of a full DI/IoC framework, and certainly DOES NOT try to compete with all DI/IoC containers out there. We aim at fulfilling only the features needed for our realtime streaming framework.
